# 时间复杂度


从图入手，每一层相当于把整个列表遍历一遍， O(n) ；总共 logn 层，所以合并也就是一半部分的时间复杂度为 O(nlogn) 。

还有因为归并排序不是” 原地排序”，我们开了一个临时空间 tmp ，所以空间复杂度为 O(n) 。
